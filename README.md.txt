# Gesture Presentation System 

## Overview
A system that allows users to control presentations using hand gestures instead of a mouse or keyboard.

## Features
- Slide navigation using gestures
- Real-time hand tracking
- Smooth presentation control

## Technologies Used
- Python
- OpenCV
- MediaPipe

## How to Run
1. Clone the repo
2. Install requirements
3. Run main.py
